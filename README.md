## Identity Infrastructure

The identity system is built around a decentralized identifier model.

Primary DID: did:web:identity.nvo987.us

Published identity artifacts include:

- identity document (`identity.json`)
- public verification key (`identity-public.pem`)
- digital signature files
- integrity hash records

These files allow external systems to verify identity metadata and signed research materials.

---

## Research Network

The infrastructure is designed to support:

- decentralized identity publication  
- research data interoperability  
- machine-readable knowledge structures  
- cryptographic verification of published artifacts  

This model enables a persistent research identity independent of centralized platforms.
