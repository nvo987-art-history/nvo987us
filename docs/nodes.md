# Infrastructure Nodes

The NVO987 infrastructure consists of multiple interconnected domains that provide specific functions within the system.


## nvo987.us

Root infrastructure domain.

https://nvo987.us

This domain acts as the central entry point of the infrastructure and references the different nodes composing the network.


## identity.nvo987.us

Identity node.

https://identity.nvo987.us

This node hosts the decentralized identifier:

did:web:identity.nvo987.us

The identity node provides:

- identity profile services
- verification endpoints
- document repositories
- machine-readable metadata
- identity infrastructure APIs


## knowledge.nvo987.us

Knowledge infrastructure node.

https://knowledge.nvo987.us

This node hosts machine-readable research resources including:

- structured research datasets
- knowledge graph data
- research publication metadata
- machine-readable knowledge representations


## lab.nvo987.us

Research laboratory node.

https://lab.nvo987.us

This environment is used for:

- experimental infrastructure development
- research prototypes
- testing decentralized systems
- experimental knowledge graph tools


## nvo987.fr

Non-profit organization associated with the infrastructure.

https://nvo987.fr

The organization supports the research and development activities associated with the NVO987 infrastructure.
