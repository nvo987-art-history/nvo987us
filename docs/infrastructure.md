# NVO987 Infrastructure

The NVO987 infrastructure is a decentralized research environment composed of multiple interconnected domains.

The root infrastructure domain is:
https://nvo987.us

The infrastructure identity is defined by the decentralized identifier:
did:web:identity.nvo987.us

## Identity Node

The identity layer of the infrastructure is hosted at:
https://identity.nvo987.us

This node publishes the DID document and provides identity-related services including:
- profile services
- document repositories
- verification services
- machine-readable metadata
- knowledge graph endpoints

## Infrastructure Domains

The infrastructure currently consists of the following domains.

### nvo987.us
Root domain of the research infrastructure.

### identity.nvo987.us
Decentralized identity node implementing:
did:web:identity.nvo987.us

### knowledge.nvo987.us
Machine-readable knowledge infrastructure hosting structured research information and datasets.

### lab.nvo987.us
Experimental research environment used for developing new infrastructure components and research tools.

### nvo987.fr
Non-profit organization associated with the infrastructure.

The infrastructure is designed to support:
- decentralized identity
- machine-readable research publication
- knowledge graph infrastructure
- AI-readable metadata
- verifiable research artifacts
